# PokéCatálogo

Projeto front-end de catálogo estilo Pokédex, desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Página inicial
- Catálogo com cards
- Busca por nome ou número
- Página de detalhes
- Navegação entre Pokémon
- Layout responsivo
- Cores por tipo

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como executar

Basta abrir o arquivo `index.html` no navegador.

Também é possível rodar com uma extensão como Live Server no VS Code.

## Estrutura

```text
.
├── index.html
├── catalogo.html
├── detalhes.html
├── css/
│   └── style.css
├── js/
│   ├── app.js
│   └── pokemon-data.js
└── README.md
```

## Observação

As imagens dos Pokémon são carregadas via sprites públicos da PokeAPI.
